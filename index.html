Wisdom Newsletter Archive

The newsletter of the UWindsor CSS, Wisdom, was first published in 1983. It contained articles written by CSS members about various computer related subjects, humour and events organized by the CSS All the old issues (that were available) were converted to HTML (hyper text markup language) and saved. Only Volumes 9, 10, and 11 were still on the system. All previous issues are lost.
